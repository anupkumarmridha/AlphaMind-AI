# Product Overview

AlphaMind AI is a self-improving agentic trading intelligence platform that combines technical analysis, event intelligence, and risk assessment to generate explainable trading decisions.

## Core Capabilities

- Multi-agent investigation system (technical, event, risk, context agents)
- Paper trading simulation with realistic slippage and commission modeling
- Continuous learning from trade outcomes to improve decision accuracy
- Explainable AI decisions with full reasoning transparency
- Real-time market data integration (price + news)

## Key Differentiators

- Agent-based architecture using LangGraph for orchestration
- Regime-dependent dynamic weighting (adjusts strategy based on market conditions)
- Hard risk veto system (blocks trades when risk is critical)
- Token-Optimized Object Notation (TOON) for efficient LLM communication
- Self-improving feedback loop (evaluation → learning → optimization)

## Target Use Cases

- Paper trading validation and strategy testing
- AI-driven market analysis and decision support
- Pattern recognition and historical performance tracking
- Educational tool for understanding multi-agent trading systems
